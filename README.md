# Персональное портфолио

[![Site preview](/public/site-preview.png)](https://webga.ru)

Мое портфолио создано с помощью [Next.js](https://nextjs.org/), [Three.js](https://threejs.org/), и [Framer Motion](https://www.framer.com/motion/). Посмотреть [сайт](https://webga.ru)

## Установка и запуск

Убедитесь, что у вас установлены nodejs `18.0.0` или выше и npm `8.6.0` или выше. Установите зависимости с помощью:

```bash
npm install
```

После этого запустите локальный сервер с помощью:

```bash
npm run dev
```

Чтобы создать производственную сборку:

```bash
npm run build
```
